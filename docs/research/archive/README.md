# Archived ONT explorations

Status: Historical archive.

Files in this folder are not active protocol specifications. They are retained
only for provenance: they show ideas ONT explored on the way to the current
design, including sponsor credits, Ark/RGB notes, public-log experiments,
subname sketches, older review briefs, and retired launch paths.

Do not infer current protocol behavior from this folder. For current design,
start with the canon homes (doc-canon (#45)):

- [`docs/README.md`](../../README.md) — the reader map
- [`docs/ONT.md`](../../ONT.md) — the plain-language source of truth
- [`docs/DESIGN.md`](../../DESIGN.md) — the full design for reviewers
- [`docs/spec/`](../../spec/) — the normative layer (e.g.
  [`ONT_ACQUISITION_STATE_MACHINE.md`](../../spec/ONT_ACQUISITION_STATE_MACHINE.md),
  [`CONTESTED_AUCTION_REFERENCE.md`](../../spec/CONTESTED_AUCTION_REFERENCE.md))
- [`docs/LAUNCH.md`](../../LAUNCH.md) — the launch narrative
- [`docs/core/STATUS.md`](../../core/STATUS.md) — what is actually wired

If an archived document disagrees with those files, the current design files win.
