# Status Logs

This directory holds daily ONT working logs.

Use one file per day:

```text
docs/status/daily/YYYY-MM-DD.md
```

Each daily log should collect:

- important decisions
- progress completed
- notable implementation or review findings
- verification results
- open questions for the ONT group
- follow-up candidates for the next work session

These logs are working notes, not polished release notes. At the end of a day,
use the current daily log to produce a concise group-chat status update.
